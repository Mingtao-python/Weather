## Quick tutorial — before viewing the example video

---

1. Open the file on GitHub and click the "Raw" button to view the file's source.  
2. The raw content will open in a new tab or window; wait a moment for it to load.  
3. Save the file using your browser’s download option (Right‑click → Save As... or press Ctrl+S / Cmd+S). The file will be downloaded to your device.

Notes:
- If the file does not open automatically, add ?raw=true to the file URL.
- Alternatively, clone the repository to access files locally.
